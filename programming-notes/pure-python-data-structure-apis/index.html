Python Data Structure API Cheatsheet

2026/06/15  ·  Programming Notes
Contents
Contents

A fast-lookup reference for the Python built-ins and collections / heapq APIs that cover ~99% of LeetCode coding interviews. Each entry is structured as:

  • Need — what you want to achieve
  • Use — which class or built-in to reach for
  • API + example — the exact call with a runnable snippet

Mental model: pick the structure by access pattern. Index/scan → list. Key lookup → dict. Membership → set. FIFO/BFS → deque. Top-k / priority → heapq. Counting → Counter. Grouping → defaultdict.

1. List

The default sequence: ordered, indexable, mutable. Doubles as a stack.

Need: create a list (empty, literal, pre-filled, 2D) — Use: list literal / comprehension

nums = [1, 2, 3]          # literal
empty = []                # empty
zeros = [0] * 5           # [0, 0, 0, 0, 0]
grid = [[0] * 3 for _ in range(2)]  # 2x3 matrix, distinct rows
# WARNING: [[0]*3]*2 shares the same inner list -> bug

Need: read / write by index — Use: list[i]

nums[0]      # first
nums[-1]     # last
nums[1] = 100  # assign

Need: slice a sub-list / reverse — Use: list[start:stop:step]

nums[1:3]    # indices 1,2
nums[:2]     # first two
nums[2:]     # from index 2 to end
nums[::-1]   # reversed copy
nums[::2]    # every other element

Need: use as a stack (push / pop end) — Use: append / pop

nums.append(4)   # push, O(1)
nums.pop()       # pop last, O(1)

Need: insert / remove at a position — Use: insert / pop(i) / remove

nums.insert(0, 9)   # insert 9 at index 0, O(n)
nums.pop(2)         # remove & return index 2, O(n)
nums.remove(9)      # remove first occurrence of value 9, O(n)
del nums[1]         # delete by index

Need: reverse / sort in place — Use: reverse / sort

nums.reverse()               # in place, no return
nums.sort()                  # ascending in place
nums.sort(reverse=True)      # descending
nums.sort(key=abs)           # by key

Need: sorted copy (keep original) — Use: sorted()

sorted(nums)                 # new ascending list
sorted(nums, reverse=True)   # new descending list

Need: length / count / find index — Use: len / count / index

len(nums)         # size
nums.count(2)     # how many 2s
nums.index(2)     # first index of value 2 (ValueError if absent)
2 in nums         # membership, O(n)

Need: aggregate — Use: sum / min / max

sum(nums)
min(nums)
max(nums)
max(nums, key=abs)   # element with max abs value

Need: extend / concatenate — Use: extend / +

nums.extend([5, 6])   # append all, in place
combined = a + b      # new list

Need: unpack head / tail or replace a slice — Use: * unpack / slice assignment

first, *rest = nums      # first=nums[0], rest=list of the rest
*init, last = nums       # last=nums[-1], init=everything before
nums[1:3] = [9, 9, 9]    # slice assignment (can change length)
nums[:] = sorted(nums)   # mutate in place while keeping same object

Need: iterate with index / two lists together — Use: enumerate / zip

for i, val in enumerate(nums):     # index + value
    ...
for i, val in enumerate(nums, 1):  # start index at 1
    ...
for a, b in zip(list1, list2):     # pairwise, stops at shorter
    ...

Need: transform / filter into a new list — Use: list comprehension

squares = [x * x for x in nums]
evens = [x for x in nums if x % 2 == 0]
flat = [v for row in grid for v in row]   # flatten 2D

2. String

Immutable sequence of characters. Any "edit" creates a new string, so build with a list + join when you need many concatenations.

Need: index / slice / reverse — Use: s[i], s[a:b], s[::-1]

s = "hello"
s[0]       # 'h'
s[-1]      # 'o'
s[1:4]     # 'ell'
s[::-1]    # 'olleh' (reverse)

Need: split into tokens / join back — Use: split / join

"a,b,c".split(",")     # ['a', 'b', 'c']
"a b  c".split()       # ['a', 'b', 'c'] (any whitespace, drops empties)
"".join(['a', 'b'])    # 'ab'
"-".join(['a', 'b'])   # 'a-b'

Need: trim / replace — Use: strip / replace

"  hi  ".strip()        # 'hi'
"xxhixx".strip("x")     # 'hi'
"aXbXc".replace("X", "") # 'abc'

Need: search within a string — Use: find / index / in

"hello".find("ll")    # 2 (returns -1 if absent)
"hello".index("ll")   # 2 (ValueError if absent)
"ll" in "hello"       # True
"hello".startswith("he")  # True
"hello".endswith("lo")    # True
"hello".count("l")        # 2

Need: classify / change case — Use: isdigit / isalpha / lower etc.

"123".isdigit()    # True
"abc".isalpha()    # True
"a1".isalnum()     # True
"Hi".lower()       # 'hi'
"Hi".upper()       # 'HI'

Need: char <-> code point — Use: ord / chr

ord('a')        # 97
chr(97)         # 'a'
ord('c') - ord('a')   # 2  (index into 26-letter alphabet)

Need: build a string efficiently — Use: list + join

parts = []
for ch in "abc":
    parts.append(ch.upper())
result = "".join(parts)   # 'ABC'  (avoid s += ch in loops)

Need: format values into text — Use: f-string

i, val = 2, 9
f"index {i} = {val}"   # 'index 2 = 9'

3. Dict

Hash map: O(1) average key lookup, insertion ordered (Python 3.7+).

Need: create / read / write — Use: {} literal, d[k]

d = {"a": 1, "b": 2}
d["a"]          # 1 (KeyError if missing)
d["c"] = 3      # insert / update

Need: read with a fallback (no KeyError) — Use: get

d.get("z")         # None if missing
d.get("z", 0)      # 0 default

Need: insert default only if absent — Use: setdefault

d.setdefault("k", []).append(1)   # init list once, then append

Need: membership / delete — Use: in / pop / del

"a" in d         # key membership, O(1)
d.pop("a")       # remove & return value
d.pop("z", None) # safe remove with default
del d["b"]       # delete key

Need: iterate keys / values / pairs — Use: keys / values / items

for k in d:               # keys
    ...
for v in d.values():      # values
    ...
for k, v in d.items():    # pairs
    ...

Need: build a dict from data — Use: dict comprehension

sq = {x: x * x for x in range(3)}        # {0:0, 1:1, 2:4}
idx = {v: i for i, v in enumerate(nums)} # value -> index

Need: ordered dict with O(1) reordering (LRU cache) — Use: OrderedDict.move_to_end

from collections import OrderedDict
lru = OrderedDict()
lru["a"] = 1
lru.move_to_end("a")          # mark as most-recently used (to the right)
lru.move_to_end("a", last=False)  # move to the left (front)
lru.popitem(last=False)       # evict least-recently used (from the left)

4. defaultdict

dict that auto-creates a default value on first access — removes if key not in d boilerplate. The argument is a factory (called with no args).

Need: group items into lists — Use: defaultdict(list)

from collections import defaultdict
groups = defaultdict(list)
for word in ["bat", "tab", "cat"]:
    key = "".join(sorted(word))
    groups[key].append(word)   # no need to init the list

Need: count occurrences — Use: defaultdict(int)

count = defaultdict(int)
for ch in "banana":
    count[ch] += 1   # starts at 0 automatically

Need: group into sets (dedupe) — Use: defaultdict(set)

adj = defaultdict(set)
adj[0].add(1)   # adjacency list for a graph

5. Counter

A dict subclass specialized for counting. Often faster to write than defaultdict(int).

Need: count everything at once — Use: Counter(iterable)

from collections import Counter
c = Counter("banana")   # {'a': 3, 'n': 2, 'b': 1}
c["a"]                  # 3
c["z"]                  # 0 (missing keys return 0, no KeyError)

Need: top-k most frequent — Use: most_common

c.most_common(2)    # [('a', 3), ('n', 2)]
c.most_common()     # all, sorted by count desc

Need: compare / combine counts — Use: arithmetic & equality

Counter("aab") == Counter("aba")   # True (anagram check)
Counter("aab") + Counter("bc")     # add counts
Counter("aab") - Counter("ab")     # subtract (keeps positive)

Need: sum of all counts — Use: total

Counter("banana").total()   # 6  (total number of items counted)

6. Set

Unordered unique elements with O(1) membership — the go-to for "have I seen this?".

Need: create (empty / from iterable) — Use: set() / {} literal

seen = set()            # empty (NOT {}, that's a dict)
s = {1, 2, 3}           # literal
s = set([1, 1, 2])      # {1, 2} from iterable

Need: add / remove / membership — Use: add / discard / in

s.add(4)        # insert
s.discard(4)    # remove if present (no error if absent)
s.remove(4)     # remove (KeyError if absent)
3 in s          # O(1) membership

Need: set algebra — Use: | & - ^

a, b = {1, 2, 3}, {2, 3, 4}
a | b   # union        {1, 2, 3, 4}
a & b   # intersection {2, 3}
a - b   # difference   {1}
a ^ b   # symmetric    {1, 4}

Need: an immutable set (e.g. dict key) — Use: frozenset

key = frozenset([1, 2])   # hashable, can be a dict key or set element

7. deque

Double-ended queue: O(1) push/pop at both ends. The correct structure for BFS (use a plain list as a queue and pop(0) is O(n) — avoid).

Need: a FIFO queue (BFS) — Use: deque + append / popleft

from collections import deque
q = deque([root])
while q:
    node = q.popleft()    # O(1) dequeue from front
    q.append(child)       # O(1) enqueue at back

Need: push / pop at the front — Use: appendleft / popleft

dq = deque([1, 2, 3])
dq.appendleft(0)   # deque([0, 1, 2, 3])
dq.popleft()       # 0
dq.pop()           # 3 (from the right)

Need: fixed-size sliding window — Use: deque(maxlen=k)

window = deque(maxlen=3)
for x in [1, 2, 3, 4]:
    window.append(x)   # auto-evicts from the left past size 3
# window -> deque([2, 3, 4], maxlen=3)

Need: rotate elements (circular shift) — Use: rotate

dq = deque([1, 2, 3, 4, 5])
dq.rotate(2)    # deque([4, 5, 1, 2, 3])  right by 2
dq.rotate(-1)   # left by 1

8. heapq

A binary min-heap built on a plain list. O(log n) push/pop, O(1) peek at the smallest. Python only has a min-heap — negate values for a max-heap.

Need: maintain a min-heap — Use: heappush / heappop

import heapq
h = []
heapq.heappush(h, 3)
heapq.heappush(h, 1)
heapq.heappop(h)   # 1 (always the smallest)
h[0]               # peek smallest without popping

Need: heapify an existing list in place — Use: heapify

h = [5, 1, 3]
heapq.heapify(h)   # O(n), now a valid min-heap

Need: a max-heap — Use: negate values

h = []
heapq.heappush(h, -val)     # push negatives
largest = -heapq.heappop(h) # negate back when popping

Need: top-k largest / smallest — Use: nlargest / nsmallest

heapq.nlargest(3, nums)             # 3 biggest
heapq.nsmallest(3, nums)            # 3 smallest
heapq.nlargest(2, items, key=len)  # by key

Need: priority queue with tie-breaking — Use: heap of tuples

h = []
heapq.heappush(h, (dist, node))   # ordered by dist, then node
d, n = heapq.heappop(h)
# tip: include a counter (dist, count, item) if items aren't comparable

Need: push+pop in one step (streaming top-k) — Use: heappushpop / heapreplace

heapq.heappushpop(h, x)   # push x, then pop & return smallest (push first)
heapq.heapreplace(h, x)   # pop & return smallest, then push x (pop first)
# keep k largest: maintain a size-k min-heap, heappushpop each new value

9. bisect

Binary search on a sorted list. O(log n) to find an insertion point, O(n) to insert. The go-to for "insert position" / "count elements < x" problems without rolling your own.

Need: find insertion index in a sorted list — Use: bisect_left / bisect_right

import bisect
a = [1, 3, 3, 5]
bisect.bisect_left(a, 3)   # 1  (leftmost spot; first index >= x)
bisect.bisect_right(a, 3)  # 3  (rightmost spot; first index > x)
bisect.bisect(a, 3)        # 3  (alias for bisect_right)

Need: insert while keeping sorted order — Use: insort

bisect.insort(a, 4)   # a -> [1, 3, 3, 4, 5]  (insort_left also exists)

Need: count / search in a sorted list — Use: bisect index math

# number of elements strictly less than x
bisect.bisect_left(a, x)
# number of elements <= x
bisect.bisect_right(a, x)
# is x present?
i = bisect.bisect_left(a, x)
present = i < len(a) and a[i] == x

10. Tuple

Immutable, hashable sequence. Great for fixed records, multiple return values, and dict/set keys (lists can't be keys).

Need: pack / unpack multiple values — Use: tuple assignment

point = (3, 4)
x, y = point          # unpack
a, b, c = 1, 2, 3     # multiple assignment

Need: swap without a temp variable — Use: tuple swap

a, b = b, a

Need: a composite key for dict / set — Use: tuple as key

seen = set()
seen.add((row, col))        # 2D coordinate as a key
visited = {(0, 0): True}    # tuple dict key

Need: return several values from a function — Use: return a tuple

def min_max(xs):
    return min(xs), max(xs)
lo, hi = min_max([3, 1, 2])

11. Sorting deep-dive

sorted() / list.sort() are stable. Control order with key and reverse.

Need: sort by a derived value — Use: key=

sorted(words, key=len)              # by length
sorted(nums, key=abs)               # by absolute value
sorted(words, key=str.lower)        # case-insensitive

Need: sort by multiple criteria — Use: key returning a tuple

# by length ascending, then alphabetically
sorted(words, key=lambda w: (len(w), w))
# by score descending, then name ascending
sorted(people, key=lambda p: (-p.score, p.name))

Need: sort dict items by value — Use: sorted(items, key=...)

sorted(d.items(), key=lambda kv: kv[1])               # by value asc
sorted(d.items(), key=lambda kv: kv[1], reverse=True) # by value desc

Need: sort a list of indices by another list — Use: key indexing

order = sorted(range(len(nums)), key=lambda i: nums[i])

12. itertools

Lazy iterator building blocks. The combinatoric ones (product, permutations, combinations) replace hand-rolled backtracking for small inputs; accumulate gives prefix sums in one line.

Need: prefix sums / running aggregate — Use: accumulate

import itertools
list(itertools.accumulate([1, 2, 3, 4]))           # [1, 3, 6, 10]  running sum
list(itertools.accumulate([3, 1, 4], max))         # [3, 3, 4]      running max
list(itertools.accumulate([1, 2, 3], initial=0))   # [0, 1, 3, 6]   with seed

Need: all r-length combinations / permutations — Use: combinations / permutations

list(itertools.combinations([1, 2, 3], 2))   # [(1,2), (1,3), (2,3)]  order doesn't matter
list(itertools.permutations([1, 2, 3], 2))   # [(1,2),(1,3),(2,1),(2,3),(3,1),(3,2)]
list(itertools.combinations_with_replacement([1, 2], 2))  # [(1,1),(1,2),(2,2)]

Need: cartesian product / nested loops flattened — Use: product

list(itertools.product([0, 1], repeat=2))    # [(0,0),(0,1),(1,0),(1,1)]  all bit patterns
list(itertools.product("ab", [1, 2]))        # [('a',1),('a',2),('b',1),('b',2)]
for r, c in itertools.product(range(rows), range(cols)):  # flat grid iteration
    ...

Need: adjacent pairs (sliding window of 2) — Use: pairwise

list(itertools.pairwise([1, 2, 3, 4]))   # [(1,2), (2,3), (3,4)]  compare neighbors

Need: concatenate / flatten iterables — Use: chain

list(itertools.chain([1, 2], [3, 4]))            # [1, 2, 3, 4]
list(itertools.chain.from_iterable(grid))        # flatten 2D rows into one stream

Need: group consecutive equal items — Use: groupby (sort first!)

for key, group in itertools.groupby(sorted(data)):
    items = list(group)   # all items equal to key
# run-length encode "aaabb": [(ch, len(list(g))) for ch, g in groupby("aaabb")]

13. Misc built-ins

Small utilities that show up constantly.

Need: generate number ranges — Use: range

range(5)         # 0..4
range(1, 5)      # 1..4
range(5, 0, -1)  # 5..1 (countdown)
range(0, 10, 2)  # 0,2,4,6,8

Need: apply / filter functionally — Use: map / filter

list(map(int, ["1", "2"]))          # [1, 2]
list(filter(lambda x: x > 0, nums)) # positives

Need: short-circuit boolean checks — Use: any / all

any(x < 0 for x in nums)   # True if any negative
all(x > 0 for x in nums)   # True if all positive

Need: quotient and remainder together — Use: divmod

q, r = divmod(17, 5)   # (3, 2)

Need: sentinel large/small values — Use: float('inf')

best = float('inf')     # for minimization
worst = float('-inf')   # for maximization

Need: parse / convert numbers — Use: int / bin / base arg

int("42")        # 42
int("1010", 2)   # 10 (binary string to int)
int("ff", 16)    # 255 (hex)
bin(5)           # '0b101'
abs(-3)          # 3

Need: reverse any iterable — Use: reversed

list(reversed([1, 2, 3]))   # [3, 2, 1]
for ch in reversed("abc"):  # 'c', 'b', 'a'
    ...

Need: transpose a matrix / unzip pairs — Use: zip(*...)

grid = [[1, 2, 3], [4, 5, 6]]
list(zip(*grid))        # [(1, 4), (2, 5), (3, 6)]  columns become rows
xs, ys = zip(*pairs)    # unzip [(1,2),(3,4)] -> (1,3), (2,4)

Need: math helpers (gcd, ceil/floor, sqrt, inf) — Use: math

import math
math.gcd(12, 8)    # 4
math.ceil(7 / 2)   # 4   (or use -(-7 // 2) to avoid float)
math.floor(7 / 2)  # 3
math.isqrt(17)     # 4   (integer sqrt, exact, no float error)
math.inf           # same idea as float('inf')
math.prod([1, 2, 3, 4])  # 24  (product, like sum but multiply)
math.comb(5, 2)    # 10  (n choose k)
math.perm(5, 2)    # 20  (n permute k)

Need: count set bits / bit tricks — Use: bin / int.bit_count

bin(13).count("1")   # 3   (popcount, works on any Python)
(13).bit_count()     # 3   (Python 3.10+, same thing)
(13).bit_length()    # 4   (number of bits to represent 13)
